# AIVoiceWebService

## О проекте

Веб-интерфейс для взаимодействия пользователя с системой нейросетевого синтеза речи. Обеспечивает ввод текста, выбор языка озвучивания, генерацию аудио и его сохранение на устройстве пользователя.

## Целевая аудитория

- **Лица с нарушениями зрения и дислексией** — преобразование текстовой информации в аудиоформат.
- **Иностранные студенты** — аудирование учебных материалов на русском и английском языках.
