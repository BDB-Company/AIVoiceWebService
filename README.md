h1. AIVoiceWebService

h2. Назначение

Веб-интерфейс для взаимодействия пользователя с системой нейросетевого синтеза речи (Text-to-Speech). Обеспечивает ввод текста, выбор языка озвучивания, генерацию аудио и его сохранение на устройстве пользователя.

h2. Целевая аудитория

* *Лица с нарушениями зрения и дислексией* — для преобразования текстовой информации в аудиоформат.
* *Иностранные студенты* — для аудирования учебных материалов на русском и английском языках.
